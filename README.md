# FLARE: Formulation-Level Automated REformulation Verification

## Building

To build the project, first run `lake update`. Then, run `lake exe cache get` to download pre-complied Mathlib. If you're new to Lean, we recommend the [Learning Lean 4](https://leanprover-community.github.io/learn.html) tutorial. The [Mathematics in Lean](https://leanprover-community.github.io/mathematics_in_lean/) is another great resource.
