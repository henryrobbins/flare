The selective graph coloring problem, also known as partition coloring, is defined on a graph whose vertex set is partitioned into clusters. The goal is to choose exactly one vertex from each cluster and color the selected vertices using as few colors as possible. A coloring is valid if any two selected vertices that are adjacent in the original graph receive different colors. The underlying graph is assumed to be perfect, meaning that for every induced subgraph the chromatic number equals the max clique number.
